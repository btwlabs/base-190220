Facebook Wall
-------------
This module allows you to fetch your Facebook wall post from your given Facebook
account.


Features
--------------------------------------------------------------------------------
* User current wall post from its Facebook profile page.
* Like of users
* Basic default profile infomation
* Filter wall post according to type of status
   * Photo
   * Link
   * Video
   * Status (text message)
* Wall Limit : get no of wall post with limit
* Enable/ Disable "Get Older Post" Button & "View Post" Link


Configuration
--------------------------------------------------------------------------------
* Go to the main settings page:
  * admin/config/services/facebook_wall
* Follow the prompts to first create a temporary access token and then a long
  term access token.

Note: Facebook removed the ability to generate a permanent access token, so it
must be renewed every sixty (60) days.

Once connected, two blocks will be made available for placement around the site:
* Facebook Page Likes
* Facebook Wall

A default page is also available that displays the wall post: faceboook_wall
